# SQRT_Project

**SQRT Members:** Orlaith Ni Dhuill, Elyana Abdallah, Javier Ayuso, Caimin Keavney, Bruce Ritter.

**Programming language:** Python (code runs on micropython controller)

**Repo Information**

This repository contains all code for the SQRT project. Older versions of code and early test data packets are included in the **old_version** branch of this repo. 

## File Directory
- **SQRT Software**: contains final software used onboard SQRT during flight ("flight software").
- **post-mission data analysis**: contains software used for post-mission data analysis, i.e. code not run onboard. Also contains collected test data
