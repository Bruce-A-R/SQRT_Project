## SOFTWARE DRAFTS
This folder contains older versions of all code written for SQRT. 

## Directory

1. **mains**: folder containing older iterations of main functions
2. **operations_classes**: folder containing older iterations of operations classes (such as helper.py, triggering_algorithm.py, servo.py)
3. **sensor_classes**: folder containing older versions of sensor classes
